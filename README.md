用于解决本科毕业论文修改周期长、格式问题多、表达重复率偏高的问题。核心痛点是学生往往不知道论文哪里不符合规范，导师反馈也比较碎片化，反复修改耗时较多。系统包含两个主要 Agent：评审 Agent 负责按学校论文模板检查摘要、目录、章节结构、图表编号、参考文献和 GB/T7714 引用格式；改写 Agent 负责在不改变原意和专业术语的前提下，对高重复段落进行学术化改写，并控制 AIGC 痕迹。逻辑流程为：用户上传论文文档后，评审 Agent 先生成问题清单，再把需要修改的段落交给改写 Agent，改写完成后再由评审 Agent 复核格式和语义一致性，最后输出修改建议、可替换文本和风险提示。项目可用于提升论文修改效率，减少人工反复检查成本，可明显提升学生论文修改通过率。
